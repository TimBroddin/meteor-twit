Twit by ttezel
==============

Node.js Twitter API client.

Repackaged for Meteorite.


Create new Twit
===============


Place this on the server side

    Twit = new TwitMaker({
        consumer_key:         '...'
      , consumer_secret:      '...'
      , access_token:         '...'
      , access_token_secret:  '...'
    });
